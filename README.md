README.md - 校园博客小程序

品牌与视觉方向

- 产品定位：面向大学生的校园内容社区，青春活力但不浮夸
- 气质关键词：清新、学院感、温暖、可读性强
- 意象锚点：午后校园草坪上的阅读时光，阳光透过树叶洒在笔记本上

Design Tokens

色彩

- 主色：#4F86C6（学院蓝）- 沉稳而不沉闷，传递信任与知识感
- 辅色：#FF8C42（暖阳橙）- 用于强调、点赞、互动热区
- 背景色：#F5F7FA（浅灰白）- 减少视觉疲劳，适合长时间阅读
- 卡片色：#FFFFFF（纯白）- 内容承载区域
- 文字主色：#2C3E50（深墨蓝）- 正文阅读
- 文字次色：#8E99A4（雾灰）- 辅助信息、时间戳
- 分割线：#EBEEF2（极浅灰）

字体

- 系统默认字体栈，不引入额外字体以控制小程序包体积
- 标题：32rpx / bold
- 正文：28rpx / regular
- 辅助文字：24rpx / regular
- 行高：1.6 倍

间距

- 页面边距：24rpx
- 卡片间距：20rpx
- 卡片内边距：24rpx
- 元素间距：16rpx

圆角

- 卡片圆角：16rpx
- 按钮圆角：40rpx（全圆角胶囊型）
- 头像圆角：50%

阴影

- 卡片阴影：0 2rpx 12rpx rgba(0,0,0,0.06)
- 浮动按钮阴影：0 4rpx 20rpx rgba(79,134,198,0.3)

交互与动效

- 点赞：心形缩放弹性动画（scale 1→1.3→1）
- 页面切换：原生左右滑动
- 下拉刷新：Vant 默认动画
- 卡片点击：轻微下沉反馈（translateY 2rpx）

设计禁忌

- 不使用大面积高饱和色块
- 不使用渐变背景（保持清爽）
- 不使用超过 3 种主色调
- 避免过于圆润的卡通风格

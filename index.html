校园博客

面向大学生的校园内容社区微信小程序，实现图文发布、互动社交、消息推送全流程功能

微信小程序 + Vant Weapp

✨ 核心功能

瀑布流首页

博客内容以瀑布流形式展示，支持下拉刷新、上拉分页加载、分类筛选，点赞动画交互

图文发布

多图上传（最多9张）、图片压缩预览、富文本编辑工具栏、发布前预览功能

互动社交

点赞、评论、回复、用户关注全套交互逻辑，实时同步互动数据

消息推送

点赞/评论/关注/系统通知分类展示，未读标记，点击已读状态切换

个人中心

用户信息展示、数据统计、博客管理、收藏/历史/草稿箱等功能入口

Vant Weapp 组件

按需引入 Vant 组件库，统一全局主题配色，实现全站视觉风格标准化

📱 页面结构

  • 🏠

    首页 - pages/index

    分类标签横向滚动 + 瀑布流博客列表 + 下拉刷新/上拉加载 + 搜索

  • 📖

    博客详情 - pages/detail

    图文内容展示 + 作者关注 + 评论区（评论/回复/点赞）+ 底部操作栏

  • ✏️

    发布页 - pages/publish

    标题输入 + 分类选择 + 内容编辑（工具栏）+ 多图上传压缩 + 预览

  • 💬

    消息页 - pages/message

    消息分类标签（全部/点赞/评论/关注/系统）+ 未读标记 + 消息列表

  • 👤

    个人中心 - pages/profile

    用户信息头部 + 数据统计 + 我的博客 + 功能菜单 + 退出登录

🚀 使用指南

安装微信开发者工具

前往 developers.weixin.qq.com 下载并安装最新版微信开发者工具

导入项目

打开开发者工具，选择"导入项目"，目录选择 miniprogram/ 文件夹

安装依赖

在 miniprogram/ 目录下执行 npm install，然后在开发者工具中点击"工具 → 构建 npm"

配置 AppID

在 project.config.json 中替换 appid 为你自己的小程序 AppID（或使用测试号）

预览调试

点击开发者工具的"编译"按钮即可预览，支持真机调试和模拟器预览
